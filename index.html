Update 0.7.9

The following major features have been added to CourseFlow:

  1. A mechanism has been added to create a publicly accessible link to a workflow. To use this, edit the workflow itself and check the box "create public page". A link to the page will appear below the workflow title. The URL of this page can be shared even with those who do not have an account.
  2. A workflow comparison tool has been added at the project level, allowing multiple workflows to be loaded and their nodes/outcomes compared.
  3. French translations now correctly appear if the url is modified from /en/ to /fr/.
